Code for:

CNNH PSS: Protein 8-class Secondary Structure Prediction by Convolutional Neural Network with Highway
APBC 2018

data_reader.py: preprocess data in matrix form and wrap in dictionary.
main.py: the main part of the program, including training and validate model.
cnn_highway.py: design the architecture of model.
layers.py: implement some layers used in cnn_highway.
